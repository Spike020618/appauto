# appauto

一个统一的命令行测试工具，支持 `pytest` 和 `playwright`。

## 安装（开发模式）

```bash
pip install -e .
```

## 使用示例

```bash
appauto run pytest --testpaths testcases/demo/test_appauto_demo.py --notify_user ou_de15ea583c7731052a0ab3bd370fc113 --case-level smoke
```
